# UNCTAD Maritime Data

This folder contains UNCTAD (United Nations Conference on Trade and Development) maritime and port statistics data.

## Data Sources

### Port Performance Data
- Container port throughput statistics
- Port efficiency indicators
- Turnaround time metrics
- Berth productivity data

### Maritime Connectivity
- Port connectivity index rankings
- Shipping network analysis
- Trade route statistics

### Data Files Structure
```
UNCTAD/
├── ports/
│   ├── container_throughput.json
│   ├── port_efficiency.json
│   └── connectivity_index.json
├── shipping/
│   ├── fleet_statistics.json
│   └── trade_routes.json
└── README.md
```

## Usage
Place UNCTAD data files in appropriate subdirectories for integration with the maritime vessel tracking platform.
